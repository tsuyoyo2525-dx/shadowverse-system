# シャドーバース風カードゲーム

カードゲーム「シャドーバース」の戦闘システムをPythonで模倣したものです。

## 機能
- 先攻・後攻のランダム決定
- マリガン（手札引き直し）
- フォロワーの召喚・攻撃
- 疾走・突進・守護などの効果
- 進化システム
- スペルカード

## 使い方
Jupyter Notebookでセルを上から順に実行してください。
最後のセルを実行するとゲームが起動します。

## 動作環境
- Python 3.x
- Jupyter Notebook

## カード一覧
| カード名 | コスト | ATK | HP | 効果 |
|--------|------|-----|-----|------|
| ダメージ魔法 | 1 | 3 | - | 全体3ダメージ |
| トラ | 1 | 1 | 1 | 突進 |
| リス | 3 | 3 | 5 | 守護 |
| クマ | 6 | 9 | 9 | なし |
